Qui suis-je ?

Je suis Sylvain Devaux, un étudiant de la Haute école Albert Jacquard dans l'option Desgin Web Transmedia.

J’ai actuellement 23 ans. Je suis un créateur web passionné, spécialisé dans la conception et le développement de sites internet. Ce que je préfère est le frontend. J'utilise principalement HTML, CSS (SCSS) et JavaScript pour réaliser mes projets, tout en me formant à l'utilisation de WordPress et de frameworks pour élargir mes compétences. J'aime travailler de manière collaborative et explorer de nouvelles idées.
